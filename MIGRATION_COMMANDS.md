## ❌ Issues & Solutions

### Issue: "Cannot find gradle"
**Cause:** Android folder not generated
**Fix:** 
```powershell
npx expo prebuild --platform android --clean
```

### Issue: "Module not found: expo"
**Cause:** Dependencies not installed
**Fix:**
```powershell
rm -r node_modules
npm install
```

### Issue: "Build failed: Gradle error"
**Cause:** Cached gradle files
**Fix:**
```powershell
rm -r node_modules,.gradle,.expo
npm install
npx expo prebuild --platform android --clean
```

### Issue: "Cannot authenticate with EAS"
**Cause:** Not logged in
**Fix:**
```powershell
eas logout
eas login
```

### Issue: "AAPT2 compilation failed"
**Cause:** Resource conflict in android/
**Fix:**
```powershell
rm -r android
npx expo prebuild --platform android --clean
```
