## 🆘 If Something Goes Wrong

| Problem | Solution |
|---------|----------|

| Push rejected | Check branch is `main` and repo exists |
| Build failed | Check Actions log for error details |
| APK not in artifacts | Build may still be running (wait 10 min) |
| Install fails on phone | Enable: Settings → Security → Unknown Sources |
