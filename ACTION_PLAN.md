Open **PowerShell** and copy-paste this:

```powershell

```

**⚠️ IMPORTANT:** Replace `YOUR_USERNAME` with your actual GitHub username

Example:
```
If your GitHub username is "john.smith":
git remote add origin https://github.com/john.smith/TKO-APP.git
```

Then press Enter and authenticate when prompted.

✅ **Done with Step 2**

### **ACTION 3: Download Your APK** (10-15 minutes)

1. Wait for GitHub Actions to build (5-10 minutes)
2. Go to: `https://github.com/YOUR_USERNAME/TKO-APP`
3. Click **Actions** tab
4. Wait for **green checkmark** ✅
5. Click the workflow run
6. Scroll to **Artifacts**
7. Download **TKO-Ground-Zero**
8. Extract `app.apk`

✅ **Done with Step 3**
